## CN2 GT vs CN2 GIA-E：这俩有什么本质区别

说清楚这个，很多选择困难就消失了。

搬瓦工的 CN2 套餐（严格来说是 CN2 GT 线路）和 CN2 GIA-E 套餐的根本区别：

1. **GT 是"半程高速"**：出境时走 CN2，到了国内骨干网以后就混入普通流量了。晚高峰可能有波动。

2. **GIA-E 是"全程高速"**：从美国机房到你家，全程走独立专线，不和大众流量抢资源，晚高峰表现稳定得多。

实测用下来，DC6 CN2 GIA-E 在晚上 9 点到 11 点这段时间，速度没有明显塌陷，这是 CN2 GT 做不到的。

搬瓦工 CN2 GIA-E 套餐还有一个优势：买了之后可以切换到 12+ 个机房，不只是 DC6。日本软银（JPOS_1）、荷兰 9929（EUNL_9）都包含在内，哪个快迁哪个，灵活度高很多。
